###### **安装步骤**

1. 拉取代码
```sh
git clone https://github.com/klren0312/walrus-library.git
```

2. 安装依赖
```sh
cd walrus-library
pnpm install
```

3. 启动开发环境
```sh
pnpm run dev
```


### 开发的架构 

电子书和封面以及书评markdown使用Walrus存储

其余数据使用Sui合约存储


### 部署

使用 Walrus Site 部署
